HTML Entities

HTML entities are used to display reserved or special characters. For example:

  • < represents the less-than sign (<)
  • > represents the greater-than sign (>)
  • & represents the ampersand (&)
  • " represents a double quotation mark (")
  • ' represents a single quotation mark (')
  •   represents a non-breaking space

To write HTML code like <a href="...">, you need to escape the characters <, >, and &.
